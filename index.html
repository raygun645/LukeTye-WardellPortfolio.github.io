University

This section showcases some of the projects i have created for University

Masters Project 1 - Procedural Levels

My first masters project was based around expanding research i had done into the area of Procedural content generation during my undergraduate disertation, for this i reseached different techniques that could be used for the puropse of level generation for a dungeon crawler game and developed them into working level generators as seen in the above video, this and my other two masters projects formed the basis for my final project, to create a dungeon crawler that uses Procedural generation and AI for gameplay elements.

Masters Project 2 - Procedural Assets and placement

My second masters project shifted focus away from procedural levels and towards procedural assets and their placement within a level, for this techniques for procedural asset placement were reseached and tested to ensure they would work within the previous level generator. Research was also conducted into how procedural assets are created and systems were developed to create procedural weapons, items and enemies.

Masters Project 3 - Artifical inteligence

The goal of my third masters project was to reseach and develop AI for the procedural enemies. Finite state machines, fuzzy state machines and behavior trees were each reseached and an acompanining system was developed, each system was then tested to see which solution worked the best based on project paramiters

Masters Project 4 - Encapsulation of previous work

For my final masters project, a prototype dungeon crawler was developed using each of the previous systems, reseach was conducted into gameplay elements found within a dungeon crawler and mechanics were developed. Once in a testable state, volunteers were invited to test the game to see if each previous system worked well together as well as seeing if the game was fun.

Personal

This section showcases some of the projects i have created as personal projects

Farmer project

One of the personal projects i'm working on currently is a game called Farmer, this is a simple game about planting crops and making money from them, each gameplay element present was prototyped using Visual scripting and then rebuilt in C++.

Above is a playlist of the current progress of the project, it is updated with new videos when features are developed and added.

Machine learning for cars

Another personal project i created was machine learning cars, the main objective for this was to simply train an agent to go around a track within Unreal engine 5, i wanted to understand more about machine learning and how it can be used wihtin games.

AI Game jam - Cooking With Words

During the AI and Games summer school a game jam was held. Myself and one other developer created a game called Cooking With words, the gameplay loop consists of the player running between stations and typing ingredients. Once the player has enough ingredients, they run back to the cooking machine which creates a cake.

In the background, each ingredient is given to a locally ran LLM which generates a name and a description for a cake, this is then passed to a locally ran stable diffusion model which generates and image for the cake using the generated name and description. This was created in 28 hours.

Mobile Project - Endless Runner

This was a smaller project i created to familiarise myself with unreal engine while trying to learn, it is an endless runner that showcases several different gameplay mechanics such as procedural obstacles, a save system for high scores and coins as well as a magnet upgrade for collecting coins easier.
